## 二、设计内容

### 2.1 题目要求

开发一个成绩管理系统，实现学生成绩管理的基本功能，包括学生、课程、成绩信息的录入、删除、查找和导入/导出等。

### 2.2 功能要求

| 序号 | 功能 | 说明 |
|------|------|------|
| 1 | 导入初始数据 | 从文件导入学生、课程、选课信息，创建成绩单 |
| 2 | 显示信息 | 分屏显示学生、课程、成绩信息 |
| 3 | 输入记录 | 键盘录入学生、课程、选课记录（含修改功能） |
| 4 | 删除记录 | 删除学生、课程、选课记录（逐条确认） |
| 5 | 查询信息 | 按学号/姓名查询学生、课程、成绩信息 |
| 6 | 批量导出数据 | 导出所有数据到时间戳命名的文件 |
| 7 | 退出系统 | 结束程序运行 |

// 学生信息结构体

## 四、程序实现

### 4.1 核心算法

#### 4.1.1 排序算法（冒泡排序）

```c
void sort_cjlist(Score cjlist[], int cjcount) {
    int i, j;
    for (i = 0; i < cjcount - 1; i++) {
        for (j = i + 1; j < cjcount; j++) {
            if (cjlist[i].zpj < cjlist[j].zpj) {
                Score temp = cjlist[i];
                cjlist[i] = cjlist[j];
                cjlist[j] = temp;
            }
        }
    }
}
```

#### 4.1.2 加权平均分计算

```c
// 加权平均分 = Σ(成绩×学分) / Σ学分
cjlist[cjcount].zpj += sclist[j].cj * clist[course_idx].xf;
cjlist[cjcount].zxf += clist[course_idx].xf;
// 最后除以总学分
if (cjlist[cjcount].zxf > 0) {
    cjlist[cjcount].zpj /= cjlist[cjcount].zxf;
}
```

#### 4.1.3 查找算法

```c
int find_student(Student slist[], int scount, char *xh) {
    int i;
    for (i = 0; i < scount; i++) {
        if (strcmp(slist[i].xh, xh) == 0) return i;
    }
    return -1;
}
```
